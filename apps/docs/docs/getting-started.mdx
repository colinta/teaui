# Getting Started

Build fullscreen terminal applications with React JSX.

## Installation

```bash
pnpm install @teaui/cli
teaui create my-app --framework react
```

## Hello World

Edit `index.tsx`:

```tsx
import React from 'react'
import {interceptConsoleLog} from '@teaui/core'
import {Box, Button, Stack, Text, run} from '@teaui/react'

// Redirect console.log away from the TUI — logs flush on exit
interceptConsoleLog()

function App() {
  return (
    <Box border="single">
      <Stack.down>
        <Text>Hello, TeaUI!</Text>
        <Button>[ OK ]</Button>
      </Stack.down>
    </Box>
  )
}

run(<App />)
```

Run it:

```bash
npx tsx index.tsx
```

Press **Ctrl-C** to exit.

## How it Works

TeaUI renders your component tree into a fullscreen terminal buffer using ["raw mode"](https://www.gnu.org/software/mit-scheme/documentation/stable/mit-scheme-ref/Terminal-Mode.html). The `run()` function:

1. Initializes the terminal in raw mode (full-screen, mouse events, etc.)
2. Creates a React reconciler that maps JSX elements to TeaUI `View` objects
3. Renders the view tree into a `Buffer`, which is flushed to the terminal
4. Handles keyboard, mouse, and window resize & focus events
5. On Ctrl-C (keybinding is configurable), restores the terminal and flushes any captured `console.log` output (`interceptConsoleLog`)

## Key Concepts

### Layout

Components receive a size constraint from their parent and report their **natural size**. `Stack` arranges children vertically or horizontally, `Box` adds padding and borders, `Space` fills available space.

### Focus & Input

Interactive components (`Button`, `Input`, `Checkbox`, etc.) can participate in focus management. Press **Tab** to cycle focus. The focused component receives keyboard events.

### Styling

Colors, borders, and text styles are configured via props. See the [Themes](/docs/themes) page for the full style system.

## Other APIs

TeaUI supports three rendering APIs:

- **React** (recommended) — JSX components via `@teaui/react`
- **Core OOP** — direct `View` instantiation via `@teaui/core` — see [Core API](/docs/core-api)
