A React-compatible terminal UI framework for building beautiful fullscreen TUI applications.

## What is TeaUI?

TeaUI lets you build fullscreen terminal applications using **React JSX** or a direct **OOP core API**. It handles rendering, input, mouse events, focus management, and layout — so you can focus on your application logic.

## Quick Start

```bash
pnpm install @teaui/core @teaui/react react @types/react
```

```tsx
import React from 'react'
import {Box, Button, Stack, run} from '@teaui/react'

function App() {
  return (
    <Box border="single">
      <Stack.down>
        Hello, TeaUI!
        <Button>[ OK ]</Button>
      </Stack.down>
    </Box>
  )
}

run(<App />)
```

```bash
npx tsx index.tsx
```
