# Progress

A progress bar. Horizontal or vertical, with optional percentage display.

<Example name="progress" title="Progress" />

## Props

| Prop          | Type                            | Default        | Description                                                        |
| ------------- | ------------------------------- | -------------- | ------------------------------------------------------------------ |
| `direction`   | `'horizontal' \| 'vertical'`    | `'horizontal'` | Bar direction                                                      |
| `min`         | `number`                        | `0`            | Minimum value                                                      |
| `max`         | `number`                        | `100`          | Maximum value                                                      |
| `value`       | `number`                        | `0`            | Current value                                                      |
| `showPercent` | `boolean`                       | `false`        | Show percentage text                                               |
| `location`    | `'left' \| 'center' \| 'right'` | `'center'`     | Where to show the percent text (`'right'` renders outside the bar) |

<details>
<summary>Core API</summary>

```js
import {Progress} from '@teaui/core'

new Progress({value: 65, max: 100, showPercent: true})
```

</details>
