# Dropdown

A dropdown selector that opens a modal picker. Supports single and multiple selection.

<Example name="dropdown" title="Dropdown" />

## Props

| Prop       | Type                 | Default | Description                   |
| ---------- | -------------------- | ------- | ----------------------------- |
| `title`    | `string`             | —       | Dropdown label                |
| `choices`  | `[string, T][]`      | —       | Array of [label, value] pairs |
| `selected` | `T \| T[]`           | —       | Currently selected value(s)   |
| `onSelect` | `(value: T) => void` | —       | Selection callback            |
| `multiple` | `boolean`            | `false` | Allow multiple selection      |

<details>
<summary>Core API</summary>

```js
import {Dropdown} from '@teaui/core'

new Dropdown({
  title: 'Color',
  choices: [
    ['Red', 'red'],
    ['Green', 'green'],
    ['Blue', 'blue'],
  ],
  selected: 'red',
  onSelect: val => console.log(val),
})
```
