# Scrollable

A scrollable container with an internal Stack for laying out children. Supports mouse wheel scrolling, scrollbars, and pinned children.

<Example name="scrollable" title="Scrollable" />

## Props

| Prop             | Type                                    | Default  | Description                                     |
| ---------------- | --------------------------------------- | -------- | ----------------------------------------------- |
| `direction`      | `'down' \| 'up' \| 'left' \| 'right'`   | `'down'` | Layout direction for children                   |
| `gap`            | `number`                                | `0`      | Gap between children                            |
| `scrollable`     | `'both' \| 'horizontal' \| 'vertical'`  | `'both'` | Which directions to allow scrolling             |
| `showScrollbars` | `boolean \| 'horizontal' \| 'vertical'` | `true`   | Show/hide scrollbars                            |
| `scrollWidth`    | `number`                                | `2`      | Columns to scroll per mouse wheel tick          |
| `keepAtBottom`   | `boolean`                               | `false`  | Auto-scroll to bottom when content grows        |
| `contentSize`    | `{width?: number, height?: number}`     | ‚Äî        | Override content size (for virtualized content) |
| `offset`         | `Point`                                 | ‚Äî        | Controlled scroll offset                        |
| `onOffsetChange` | `(offset: Point) => void`               | ‚Äî        | Called when scroll offset changes               |
| `children`       | `ReactNode`                             | ‚Äî        | Children laid out by the internal Stack         |

### Static Constructors

Use `Scrollable.down`, `Scrollable.up`, `Scrollable.left`, `Scrollable.right` to set the direction without the `direction` prop.

<details>
<summary>Core API</summary>

```js
import {Scrollable, Text} from '@teaui/core'

Scrollable.down(
  Array.from({length: 50}, (_, i) => new Text({text: `Line ${i + 1}`})),
  {height: 10},
)
```
