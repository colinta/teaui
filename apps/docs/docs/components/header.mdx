# Header

Large text header with border decoration. Available as `H1` through `H6` in React.

<Example name="header" title="Header" />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | — | Header text |
| `border` | `'single' \| 'bold' \| 'double'` | `'single'` | Border style |
| `font` | `FontFamily` | — | Text font |
| `bold` | `boolean` | — | Bold text |
| `dim` | `boolean` | — | Dimmed text |

React shorthand: `<H1>` = `border="double"`, `<H2>` = `border="bold"`, `<H3>`–`<H6>` = `border="single"` with decreasing prominence.
