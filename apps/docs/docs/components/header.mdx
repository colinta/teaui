# Header

Large text header with underline decoration. Available as `H1` through `H6` in React.

<Example name="header" title="Header" />

## Props

| Prop     | Type                                       | Default    | Description     |
| -------- | ------------------------------------------ | ---------- | --------------- |
| `text`   | `string`                                   | ‚Äî          | Header text     |
| `border` | `'none' \| 'single' \| 'bold' \| 'double'` | `'single'` | Underline style |
| `font`   | `FontFamily`                               | ‚Äî          | Text font       |
| `bold`   | `boolean`                                  | ‚Äî          | Bold text       |
| `dim`    | `boolean`                                  | ‚Äî          | Dimmed text     |

React shorthand: `<H1>` = `bold + border="double"`, `<H2>` = `bold + border="bold"`, `<H3>` = `bold + border="single"`, `<H4>` = `border="single"`, `<H5>` = `dim + border="single"`, `<H6>` = `dim + border="none"`.
