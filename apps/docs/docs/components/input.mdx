# Input

Text input field with cursor, selection, word movement (Alt+←→), clipboard support, and optional multiline/wrapping.

<TerminalScreenshot name="input" title="Input" />

## React Example

```tsx
import React, {useState} from 'react'
import {Stack, Input, Text, run} from '@teaui/react'

function App() {
  const [name, setName] = useState('')

  return (
    <Stack.down>
      <Text>Enter your name:</Text>
      <Input
        value={name}
        placeholder="Type here..."
        onChange={setName}
        onSubmit={(val) => console.log('submitted:', val)}
      />
    </Stack.down>
  )
}

run(<App />)
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | `''` | Current text |
| `placeholder` | `string` | — | Placeholder when empty |
| `onChange` | `(text: string) => void` | — | Text change callback |
| `onSubmit` | `(text: string) => void` | — | Enter key callback |
| `wrap` | `boolean` | `false` | Wrap long lines |
| `multiline` | `boolean` | `false` | Allow multiple lines |
| `font` | `FontFamily` | `'default'` | Text font |
