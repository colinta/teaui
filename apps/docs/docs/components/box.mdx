# Box

A container with optional border and padding.

<Example name="box" title="Box" />

## Props

| Prop        | Type                                                                            | Default    | Description                                  |
| ----------- | ------------------------------------------------------------------------------- | ---------- | -------------------------------------------- |
| `border`    | `'none' \| 'single' \| 'bold' \| 'double' \| 'rounded' \| 'dotted'` | `'single'` | Border style (or custom `BorderChars` tuple) |
| `highlight` | `boolean`                                                           | `false`    | Use highlight style for the border           |
| `padding`   | `number \| {top, bottom, left, right}`                              | `0`        | Inner padding (inherited from View)          |
| `children`  | `ReactNode`                                                         | —          | Child views                                  |
