# Box

A container with optional border and padding.

<Example name="box" title="Box" />

## Props

| Prop       | Type                                                  | Default  | Description   |
| ---------- | ----------------------------------------------------- | -------- | ------------- |
| `border`   | `'single' \| 'bold' \| 'double' \| 'round' \| 'none'` | `'none'` | Border style  |
| `padding`  | `number \| {top, bottom, left, right}`                | `0`      | Inner padding |
| `children` | `ReactNode`                                           | —        | Child views   |
