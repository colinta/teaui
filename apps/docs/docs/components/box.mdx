# Box

A styled container with optional border, padding, and background color. The fundamental layout building block.

<TerminalScreenshot name="box" title="Box" />

## React Example

```tsx
import React from 'react'
import {Box, Text, run} from '@teaui/react'

function App() {
  return (
    <Box border="single">
      <Text>Hello from a box!</Text>
    </Box>
  )
}

run(<App />)
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `border` | `'none' \| 'single' \| 'bold' \| 'double' \| 'rounded' \| 'dotted' \| 'popout' \| BorderChars` | `'none'` | Border style |
| `highlight` | `boolean` | `false` | Highlighted border appearance |
| `children` | `ReactNode` | — | Child content |

Inherits all View props (`width`, `height`, `flex`, etc.)
