# Collapsible

Toggleable container that shows either a collapsed or expanded view. When `showCollapsed` is true, both views are shown when expanded.

<Example name="collapsible" title="Collapsible" />

## Props

| Prop            | Type        | Default | Description                            |
| --------------- | ----------- | ------- | -------------------------------------- |
| `isCollapsed`   | `boolean`   | `true`  | Collapsed state                        |
| `showCollapsed` | `boolean`   | `false` | Show collapsed view even when expanded |
| `collapsed`     | `ReactNode` | —       | View when collapsed                    |
| `expanded`      | `ReactNode` | —       | View when expanded                     |

<details>
<summary>Core API</summary>

```js
import {Collapsible, Text, Button} from '@teaui/core'

new Collapsible({
  isCollapsed: true,
  collapsed: new Button({title: 'Show more'}),
  expanded: new Text({text: 'Full content'}),
})
```

</details>
