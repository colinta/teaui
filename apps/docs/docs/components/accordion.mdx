# Accordion

Expandable/collapsible sections. Click to toggle. Child sections must be `<Accordion.Section>` components.

If `multiple` is `true`, multiple sections can be open at once. Otherwise, opening one section will close other open section.

<Example name="accordion" title="Accordion" />

## Props

### Accordion

| Prop       | Type                  | Default | Description                  |
| ---------- | --------------------- | ------- | ---------------------------- |
| `multiple` | `boolean`             | `false` | Allow multiple open sections |
| `children` | `Accordion.Section[]` | —       | Section children             |

### Accordion.Section

| Prop       | Type        | Default | Description         |
| ---------- | ----------- | ------- | ------------------- |
| `title`    | `string`    | —       | Section header text |
| `isOpen`   | `boolean`   | `false` | Initially open      |
| `children` | `ReactNode` | —       | Section content     |
