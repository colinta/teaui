# Accordion

Expandable/collapsible sections. Click or press Enter to toggle. Supports single or multiple open sections.

<TerminalScreenshot name="accordion" title="Accordion" />

## React Example

```tsx
import React from 'react'
import {Accordion, Text, run} from '@teaui/react'

function App() {
  return (
    <Accordion>
      <Accordion.Section title="Section 1" isOpen>
        <Text>Content for section 1</Text>
      </Accordion.Section>
      <Accordion.Section title="Section 2">
        <Text>Content for section 2</Text>
      </Accordion.Section>
      <Accordion.Section title="Section 3">
        <Text>Content for section 3</Text>
      </Accordion.Section>
    </Accordion>
  )
}

run(<App />)
```

## Props

### Accordion

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `multiple` | `boolean` | `false` | Allow multiple open sections |
| `children` | `Accordion.Section[]` | — | Section children |

### Accordion.Section

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | — | Section header text |
| `isOpen` | `boolean` | `false` | Initially open |
| `children` | `ReactNode` | — | Section content |
