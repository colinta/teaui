# Spinner

An animated loading spinner. Animates when `isAnimating` is true.

<Example name="spinner" title="Spinner" />

## Props

| Prop          | Type      | Default | Description          |
| ------------- | --------- | ------- | -------------------- |
| `isAnimating` | `boolean` | `false` | Start/stop animation |

The spinner is 1×1 cell and cycles through braille animation frames.

<details>
<summary>Core API</summary>

```js
import {Spinner} from '@teaui/core'

new Spinner({isAnimating: true})
```

</details>
