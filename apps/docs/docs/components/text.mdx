# Text

Renders styled text. Supports alignment, wrapping, and custom fonts.

<Example name="text" title="Text" />

In React, you can also use plain strings as children of `Stack` and `Box` тАФ they're automatically wrapped in `Text`.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `text` | `string` | тАФ | Text content |
| `lines` | `string[]` | тАФ | Pre-split lines (alt to `text`) |
| `alignment` | `'left' \| 'center' \| 'right'` | `'left'` | Text alignment |
| `wrap` | `boolean` | `false` | Wrap long lines |
| `font` | `FontFamily` | `'default'` | Text font |
| `style` | `Style` | тАФ | Text style (color, bold, etc.) |
