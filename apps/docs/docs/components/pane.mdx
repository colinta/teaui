# Pane

A split-pane container for index/detail layouts. The first N-1 children are
collapsible "browser" panes with draggable separators. The last child is the
"detail" pane that takes remaining space.

## React Example

```tsx
import React from 'react'
import {Pane, Text, Stack, run} from '@teaui/react'

function App() {
  return (
    <Pane border>
      <Stack.down>
        <Text>Inbox</Text>
        <Text>Drafts</Text>
        <Text>Sent</Text>
      </Stack.down>
      <Text>Select an item to view details.</Text>
    </Pane>
  )
}

run(<App />)
```

## Props

| Prop          | Type      | Default | Description                                                                                         |
| ------------- | --------- | ------- | --------------------------------------------------------------------------------------------------- |
| `border`      | `boolean` | `false` | Draw a border around the pane                                                                       |
| `collapsible` | `boolean` | `true`  | Whether browser panes can be collapsed. When `false`, min pane width is 4 and collapse is disabled. |

Plus all standard `ViewProps` (flex, width, height, padding, theme, etc.)

## Interactions

- **Drag** a separator to resize the adjacent browser pane.
- **Click** a separator to collapse or expand its browser pane (when `collapsible`).

## Multiple Browser Panes

You can have multiple browser panes. All children except the last are browser
panes with their own separator:

```tsx
<Pane border>
  <Stack.down>
    <Text>Categories</Text>
  </Stack.down>
  <Stack.down>
    <Text>Items</Text>
  </Stack.down>
  <Text>Detail view</Text>
</Pane>
```
