# Tree

Expandable tree view. Renders hierarchical data with expand/collapse toggles and tree-line decorations.

<Example name="tree" title="Tree" />

## Props

| Prop          | Type                            | Default | Description                          |
| ------------- | ------------------------------- | ------- | ------------------------------------ |
| `data`        | `T[]`                           | —       | Root data items                      |
| `render`      | `(item: T) => ReactNode`        | —       | Render function for each node        |
| `getChildren` | `(item: T) => T[] \| undefined` | —       | Child accessor                       |
| `title`       | `ReactNode \| string`           | —       | Root title (string or React element) |

<details>
<summary>Core API</summary>

```js
import {Tree, Text} from '@teaui/core'

new Tree({
  data: [{name: 'src'}, {name: 'package.json'}],
  titleView: new Text({text: 'Project'}),
  render: item => new Text({text: item.name}),
  getChildren: item => item.children,
})
```

</details>
