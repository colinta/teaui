# ScrollableList

Virtualized scrollable list. Only renders visible rows, making it efficient for large datasets.

:::note Core API Only
ScrollableList is not available as a React component because `renderItem` returns core `View` instances, not React elements. Use `Scrollable` with React children for React apps, or use ScrollableList directly via the core API for large virtualized lists.
:::

## React Example

```tsx
import React from 'react'
import {ScrollableList, Text, run} from '@teaui/react'

const items = Array.from({length: 1000}, (_, i) => `Item ${i + 1}`)

function App() {
  return (
    <ScrollableList
      height={15}
      items={items}
      renderItem={item => new Text({text: item})}
    />
  )
}

run(<App />)
```

## Props

| Prop             | Type                             | Default | Description                             |
| ---------------- | -------------------------------- | ------- | --------------------------------------- |
| `items`          | `T[]`                            | —       | Data array                              |
| `renderItem`     | `(item: T, row: number) => View` | —       | View factory for each item              |
| `showScrollbars` | `boolean`                        | `true`  | Show/hide scrollbars                    |
| `scrollHeight`   | `number`                         | `1`     | Rows per mouse wheel tick               |
| `keepAtBottom`   | `boolean`                        | `false` | Auto-scroll to bottom (useful for logs) |
