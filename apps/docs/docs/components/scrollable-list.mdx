# ScrollableList

Virtualized scrollable list. Only renders visible rows, making it efficient for large datasets.

<Example name="scrollable-list" title="ScrollableList" />

## Props

| Prop             | Type                             | Default | Description                             |
| ---------------- | -------------------------------- | ------- | --------------------------------------- |
| `items`          | `T[]`                            | ‚Äî       | Data array                              |
| `renderItem`     | `(item: T, row: number) => View` | ‚Äî       | View factory for each item              |
| `showScrollbars` | `boolean`                        | `true`  | Show/hide scrollbars                    |
| `keepAtBottom`   | `boolean`                        | `false` | Auto-scroll to bottom (useful for logs) |

<details>
<summary>Core API</summary>

```js
import {ScrollableList, Text} from '@teaui/core'

const items = Array.from({length: 1000}, (_, i) => `Item ${i + 1}`)

new ScrollableList({
  height: 15,
  items,
  renderItem: item => new Text({text: item}),
})
```
