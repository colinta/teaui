# Window

The root container. Fills the entire terminal. Typically the outermost element of your app, created by `run()` or `Screen.start()`. You don't really need to think about or use this one.

## React Example

```tsx
import React from 'react'
import {Box, Text, run} from '@teaui/react'

function App() {
  ...
}

// run() automatically wraps your component in a Window
run(<App />)
```

## Props

| Prop       | Type        | Description   |
| ---------- | ----------- | ------------- |
| `children` | `ReactNode` | Child content |

Window always fills available space. It inherits from Container.

<details>
<summary>Core API</summary>

```js
import {Screen, Window, Box, Text} from '@teaui/core'

Screen.start(
  new Window({
    child: new Box({
      border: 'single',
      children: [new Text({text: 'Full screen app'})],
    }),
  }),
)
```
