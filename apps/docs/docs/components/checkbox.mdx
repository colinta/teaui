# Checkbox

A togglable checkbox with label text.

<Example name="checkbox" title="Checkbox" />

## Props

| Prop       | Type                       | Default | Description         |
| ---------- | -------------------------- | ------- | ------------------- |
| `title`    | `string`                   | —       | Label text          |
| `value`    | `boolean`                  | `false` | Checked state       |
| `onChange` | `(value: boolean) => void` | —       | Called when toggled |
| `hotKey`   | `HotKey`                   | —       | Keyboard shortcut   |
