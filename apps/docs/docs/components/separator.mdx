# Separator

A horizontal or vertical divider line.

<TerminalScreenshot name="separator" title="Separator" />

## React Example

```tsx
import React from 'react'
import {Stack, Text, Separator, run} from '@teaui/react'

function App() {
  return (
    <Stack.down>
      <Text>Above</Text>
      <Separator direction="horizontal" />
      <Text>Below</Text>
    </Stack.down>
  )
}

run(<App />)
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `direction` | `'horizontal' \| 'vertical'` | — | Orientation (required) |
| `padding` | `number` | `0` | Padding on each side |
| `border` | `'single' \| 'leading' \| 'trailing' \| 'bold' \| 'dash' \| 'dash2' \| 'dash3' \| 'dash4' \| 'double'` | `'single'` | Line style |

Static constructors: `Separator.horizontal()`, `Separator.vertical()`

<details>
<summary>Core API</summary>

import {Stack, Text, Separator} from '@teaui/core'

Stack.down([
