# Stack

Arranges children vertically or horizontally. The primary layout component.

<Example name="stack" title="Stack" />

## Direction Shortcuts

- `<Stack.down>` — top to bottom (default)
- `<Stack.up>` — bottom to top
- `<Stack.right>` — left to right
- `<Stack.left>` — right to left

## Props

| Prop        | Type                                  | Default  | Description              |
| ----------- | ------------------------------------- | -------- | ------------------------ |
| `direction` | `'up' \| 'down' \| 'left' \| 'right'` | `'down'` | Layout direction         |
| `gap`       | `number`                              | `0`      | Spacing between children |
| `fill`      | `boolean`                             | `true`   | Fill available space     |
| `children`  | `ReactNode`                           | —        | Child views              |
