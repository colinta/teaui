# Drawer

A slide-out panel from any edge (left, right, top, bottom). Contains a drawer view and a content view.

<Example name="drawer" title="Drawer" />

## Props

| Prop       | Type                                     | Default  | Description                 |
| ---------- | ---------------------------------------- | -------- | --------------------------- |
| `location` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'left'` | Edge to slide from          |
| `isOpen`   | `boolean`                                | `false`  | Open/closed state           |
| `onToggle` | `(isOpen: boolean) => void`              | —        | Toggle callback             |
| `drawer`   | `ReactNode`                              | —        | Drawer content              |
| `hotKey`   | `HotKey`                                 | —        | Keyboard shortcut to toggle |
| `content`  | `ReactNode`                              | —        | Main content                |

<details>
<summary>Core API</summary>

```js
import {Drawer, Text} from '@teaui/core'

new Drawer({
  location: 'left',
  drawer: new Text({text: 'Sidebar'}),
  content: new Text({text: 'Main'}),
})
```
