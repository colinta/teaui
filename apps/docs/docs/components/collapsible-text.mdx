# CollapsibleText

A text block that truncates to a single line when collapsed. Click to expand and show the full text.

<Example name="collapsible-text" title="CollapsibleText" />

## Props

| Prop    | Type     | Default | Description  |
| ------- | -------- | ------- | ------------ |
| `text`  | `string` | —       | Text content |
| `style` | `Style`  | —       | Text style   |

Click to toggle between collapsed (1 line) and expanded (full text).

<details>
<summary>Core API</summary>

```js
import {CollapsibleText} from '@teaui/core'

new CollapsibleText({text: 'Long text content...'})
```

</details>
