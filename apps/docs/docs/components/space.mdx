# Space

Empty space filler. Use to add gaps or push content apart in a Stack.

<TerminalScreenshot name="space" title="Space" />

## React Example

```tsx
import React from 'react'
import {Stack, Text, Space, run} from '@teaui/react'

function App() {
  return (
    <Stack.right>
      <Text>Left</Text>
      <Space />
      <Text>Right</Text>
    </Stack.right>
  )
}

run(<App />)
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `background` | `Color` | — | Background color fill |
| `width` | `Dimension` | — | Fixed or flex width |
| `height` | `Dimension` | — | Fixed or flex height |

Static constructors: `Space.horizontal(size)`, `Space.vertical(size)`

<details>
<summary>Core API</summary>

```js
import {Stack, Text, Space} from '@teaui/core'

const layout = Stack.right([
  new Text({text: 'Left'}),
  new Space(),

])
```
